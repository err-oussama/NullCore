# Executable & Linkable Format 




## Main Component of ELF

### ELF Header 
contains metadata about the file, such as its type, architecture, and entry point.



### Program Header Table
Holds metadata about the segment used at run time (like code and data)



### Section Header Table

Holds metadata of section that used for linking and debugging (like symbol tables and relocation information).


### Sections and Segments 

the actual data, such as code, data, symboles, and dubug information.
